# Visualizations

This directory will contain chart configurations, Jupyter notebooks, and static visual assets that accompany newsletter articles.

## Planned Content

| Type | Description |
|------|-------------|
| Jupyter Notebooks (`.ipynb`) | Python-based analysis and chart generation code |
| Chart configs (`.json`) | Vega-Lite or similar declarative chart specifications |
| Exported images (`.png`, `.svg`) | Final chart images used in published articles |

## Naming Convention

Files should be named using the pattern:
```
YYYY-MM_topic-description.ext
```

For example:
- `2025-03_uganda-mobile-money-growth.ipynb`
- `2025-03_east-africa-gdp-comparison.png`

## Requirements

To run Jupyter notebooks in this directory:
```bash
pip install pandas matplotlib seaborn jupyter
```

This directory is currently empty and will be populated as newsletter articles are published.
