# Africa-Wide Data

This directory contains datasets that span multiple African countries or cover Sub-Saharan Africa as a whole.

## Subdirectories

| Folder | Contents |
|--------|----------|
| `economy/` | Aggregate GDP and economic indicators |
| `technology/` | Mobile money and digital access across African countries |

## Primary Sources

- **World Bank Open Data:** https://data.worldbank.org
- **World Bank Global Findex:** https://globalfindex.worldbank.org
- **African Development Bank:** https://dataportal.opendataforafrica.org

## Notes

- Data coverage varies by country; refer to individual dataset files for specifics.
- Preference for datasets that include Uganda and East African countries, which are the primary focus of this newsletter.
