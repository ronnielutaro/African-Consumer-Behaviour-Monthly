# Africa-Wide — Technology

Digital and fintech adoption datasets across African countries.

## Files

| File | Description | Period | Source |
|------|-------------|--------|--------|
| `mobile_money_adoption_africa.csv` | Mobile money account ownership (% adults 15+) for selected African countries | 2011–2021 | World Bank Global Findex Database |

## Column Reference

### `mobile_money_adoption_africa.csv`
| Column | Unit | Description |
|--------|------|-------------|
| `year` | YYYY | Survey year |
| `country_name` | text | Full country name in English |
| `country_code` | ISO 3166-1 alpha-3 | 3-letter country code |
| `mobile_money_account_pct_adults_15plus` | % | Adults (15+) reporting a mobile money account |
| `survey_name` | text | Global Findex survey round |

## Countries Covered
Cameroon, Côte d'Ivoire, Ethiopia, Ghana, Kenya, Nigeria, Rwanda, Senegal, South Africa, Tanzania, Uganda, Zambia, Zimbabwe

## Notes
- Data is from the Global Findex survey, conducted every 3 years.
- Country selection prioritizes East Africa and major Sub-Saharan African economies.
- Additional countries will be added in future updates.
