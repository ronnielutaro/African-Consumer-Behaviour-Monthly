# Africa-Wide — Economy

Aggregate economic datasets covering Sub-Saharan Africa.

## Files

| File | Description | Period | Source |
|------|-------------|--------|--------|
| `gdp_sub_saharan_africa_annual.csv` | Sub-Saharan Africa total GDP (current USD) | 2000–2023 | World Bank (NY.GDP.MKTP.CD — Sub-Saharan Africa aggregate) |

## Column Reference

### `gdp_sub_saharan_africa_annual.csv`
| Column | Unit | Description |
|--------|------|-------------|
| `year` | YYYY | Calendar year |
| `sub_saharan_africa_gdp_current_usd` | USD | Total GDP for Sub-Saharan Africa at current prices |
| `sub_saharan_africa_gdp_growth_pct` | % | Annual GDP growth rate for the region |
