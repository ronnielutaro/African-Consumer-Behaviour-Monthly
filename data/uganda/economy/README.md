# Uganda — Economy

Datasets covering Uganda's macroeconomic performance.

## Files

| File | Description | Period | Source |
|------|-------------|--------|--------|
| `gdp_annual.csv` | Annual GDP (current USD) and GDP per capita | 2000–2023 | World Bank (NY.GDP.MKTP.CD, NY.GDP.PCAP.CD) |
| `inflation_annual.csv` | Annual consumer price inflation (%) | 2000–2023 | World Bank (FP.CPI.TOTL.ZG) |

## Column Reference

### `gdp_annual.csv`
| Column | Unit | Description |
|--------|------|-------------|
| `year` | YYYY | Calendar year |
| `gdp_current_usd` | USD | Gross Domestic Product at current prices |
| `gdp_per_capita_current_usd` | USD | GDP divided by mid-year population |
| `gdp_growth_pct` | % | Annual GDP growth rate |

### `inflation_annual.csv`
| Column | Unit | Description |
|--------|------|-------------|
| `year` | YYYY | Calendar year |
| `inflation_cpi_pct` | % | Annual change in Consumer Price Index |
