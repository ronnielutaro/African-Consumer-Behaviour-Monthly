# Uganda — Demographics

Datasets on Uganda's population and urbanization.

## Files

| File | Description | Period | Source |
|------|-------------|--------|--------|
| `population_annual.csv` | Total population and urban/rural share | 2000–2023 | World Bank (SP.POP.TOTL, SP.URB.TOTL.IN.ZS) |

## Column Reference

### `population_annual.csv`
| Column | Unit | Description |
|--------|------|-------------|
| `year` | YYYY | Calendar year |
| `population_total` | persons | Mid-year estimate of total population |
| `urban_population_pct` | % | Share of population living in urban areas |
| `rural_population_pct` | % | Share of population living in rural areas |
