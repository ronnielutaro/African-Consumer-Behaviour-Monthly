# Uganda — Technology

Datasets on digital access, mobile connectivity, and fintech adoption in Uganda.

## Files

| File | Description | Period | Source |
|------|-------------|--------|--------|
| `internet_penetration_annual.csv` | Internet users as % of population | 2000–2022 | World Bank / ITU (IT.NET.USER.ZS) |
| `mobile_subscriptions_annual.csv` | Mobile cellular subscriptions per 100 people | 2000–2022 | World Bank / ITU (IT.CEL.SETS.P2) |
| `mobile_money_annual.csv` | Mobile money account ownership (% adults 15+) | 2014–2021 | World Bank Global Findex Database |

## Column Reference

### `internet_penetration_annual.csv`
| Column | Unit | Description |
|--------|------|-------------|
| `year` | YYYY | Calendar year |
| `internet_users_pct_population` | % | Individuals using the internet as % of total population |

### `mobile_subscriptions_annual.csv`
| Column | Unit | Description |
|--------|------|-------------|
| `year` | YYYY | Calendar year |
| `mobile_subscriptions_per_100_people` | subscriptions per 100 | Active mobile-cellular telephone subscriptions per 100 inhabitants |

### `mobile_money_annual.csv`
| Column | Unit | Description |
|--------|------|-------------|
| `year` | YYYY | Survey year |
| `mobile_money_account_pct_adults_15plus` | % | Adults (15+) who report having a mobile money account |
| `survey_name` | text | Name of the Global Findex survey round |

## Notes
- Mobile money data is collected via household survey every three years (2011, 2014, 2017, 2021). Values for intervening years are not available.
- Internet penetration figures include both mobile and fixed-line internet access.
