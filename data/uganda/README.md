# Uganda Data

This directory contains datasets specific to Uganda, organized by topic.

## Subdirectories

| Folder | Contents |
|--------|----------|
| `economy/` | GDP, inflation, trade, and fiscal indicators |
| `demographics/` | Population, urbanization, and household survey data |
| `technology/` | Internet access, mobile subscriptions, and mobile money adoption |
| `business/` | Business registrations, investment, and sectoral data |

## Primary Sources

- **Uganda Bureau of Statistics (UBOS):** https://www.ubos.org
- **World Bank Open Data:** https://data.worldbank.org
- **Bank of Uganda:** https://www.bou.or.ug
- **Uganda Investment Authority (UIA):** https://www.ugandainvest.go.ug
- **International Telecommunication Union (ITU):** https://www.itu.int/en/ITU-D/Statistics

## Notes

- All monetary values are in current USD unless otherwise noted in the file
- Population figures are mid-year estimates
- Mobile money data from World Bank Global Findex is survey-based and collected every three years
