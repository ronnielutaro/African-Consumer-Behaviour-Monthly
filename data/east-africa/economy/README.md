# East Africa — Economy

Comparative macroeconomic datasets for EAC member states.

## Files

| File | Description | Period | Source |
|------|-------------|--------|--------|
| `gdp_comparison_annual.csv` | Annual GDP (current USD) by country | 2000–2023 | World Bank |
| `gdp_per_capita_comparison_annual.csv` | Annual GDP per capita (current USD) by country | 2000–2023 | World Bank |

## Column Reference

### `gdp_comparison_annual.csv` and `gdp_per_capita_comparison_annual.csv`
| Column | Unit | Description |
|--------|------|-------------|
| `year` | YYYY | Calendar year |
| `country_name` | text | Full country name in English |
| `country_code` | ISO 3166-1 alpha-3 | 3-letter country code |
| `gdp_current_usd` | USD | Gross Domestic Product at current prices |
| `gdp_per_capita_current_usd` | USD | GDP divided by mid-year population |
