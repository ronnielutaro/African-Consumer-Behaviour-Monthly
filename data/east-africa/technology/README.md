# East Africa — Technology

Comparative digital access and fintech adoption datasets for EAC member states.

## Files

| File | Description | Period | Source |
|------|-------------|--------|--------|
| `internet_penetration_comparison_annual.csv` | Internet users (% of population) by country | 2005–2022 | World Bank / ITU |
| `mobile_money_comparison.csv` | Mobile money account ownership (% adults 15+) by country | 2011–2021 | World Bank Global Findex |

## Column Reference

### `internet_penetration_comparison_annual.csv`
| Column | Unit | Description |
|--------|------|-------------|
| `year` | YYYY | Calendar year |
| `country_name` | text | Full country name in English |
| `country_code` | ISO 3166-1 alpha-3 | 3-letter country code |
| `internet_users_pct_population` | % | Individuals using the internet as % of total population |

### `mobile_money_comparison.csv`
| Column | Unit | Description |
|--------|------|-------------|
| `year` | YYYY | Survey year |
| `country_name` | text | Full country name in English |
| `country_code` | ISO 3166-1 alpha-3 | 3-letter country code |
| `mobile_money_account_pct_adults_15plus` | % | Adults (15+) with a mobile money account |
| `survey_name` | text | Global Findex survey round |
