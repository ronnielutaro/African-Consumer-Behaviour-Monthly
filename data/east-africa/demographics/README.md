# East Africa — Demographics

Population datasets for EAC member states.

## Files

| File | Description | Period | Source |
|------|-------------|--------|--------|
| `population_comparison_annual.csv` | Total population by country | 2000–2023 | World Bank (SP.POP.TOTL) |

## Column Reference

### `population_comparison_annual.csv`
| Column | Unit | Description |
|--------|------|-------------|
| `year` | YYYY | Calendar year |
| `country_name` | text | Full country name in English |
| `country_code` | ISO 3166-1 alpha-3 | 3-letter country code |
| `population_total` | persons | Mid-year estimate of total population |
